🔑 API Token
💻 Upload via curl
curl -H "X-Upload-Token: YOUR_TOKEN" \
  -F "file=@myfile.pdf" \
  /api/upload
## List uploaded files
curl /api/files

## Download a file
curl -o "filename.pdf" \
  "/api/files?file=filename.pdf"

## Delete a file
curl -X DELETE \
  -H "X-Upload-Token: YOUR_TOKEN" \
  "/api/files?file=filename.pdf"
